# valet-plus

About

These drivers extend the Valet+ Magento drivers allowing Magento and WordPress to work together using the FishPig integration

Installation

- Copy these files into ~/.valet/Drivers/

Requirements

- Magento / Magento2
- WordPress
- FishPig integration module
